## 1. Como navegar

Cada arquivo `.md` é independente e traz:
- 📑 um **sumário (TOC)** com links internos;
- 🧱 seções numeradas com **tabelas** e **callouts** (💡 curiosidades, ⚠️ alertas, 📌 destaques);
- 🖼️ **imagens** ilustrativas (Wikimedia Commons);
- 🔗 uma seção **"Veja também"** que conecta os temas.

> 🧠 Os conteúdos se cruzam: a arquitetura **x86**, por exemplo, aparece no Datapoint 2200, no IBM PC, nos processadores Intel e até no notebook pessoal — seguindo os links você percorre a história inteira.
