## 1. O que é o Raspberry Pi

O **Raspberry Pi** é um **computador de placa única (SBC — *Single-Board Computer*)**: toda a máquina (processador, memória, vídeo, USB, rede) cabe em uma placa do tamanho de um cartão de crédito. Diferente do Arduino (um microcontrolador), o Pi é um **computador de verdade** que roda um **sistema operacional** completo (geralmente **Linux / Raspberry Pi OS**).

---

<a id="2-origem-e-propósito"></a>
## 2. Origem e propósito

- 🇬🇧 Criado pela **Raspberry Pi Foundation** (Reino Unido).
- 📅 Primeiro modelo lançado em **2012**.
- 🎓 Objetivo inicial: **ensinar programação e ciência da computação** nas escolas, com hardware barato.
- 🌍 Tornou-se um fenômeno mundial entre **makers, hobbistas e a indústria**, com dezenas de milhões de unidades vendidas.

<a id="3-raspberry-pi-5-a-placa-mais-avançada"></a>
## 3. Raspberry Pi 5 a placa mais avançada

A **Raspberry Pi 5** trouxe ganhos expressivos de desempenho:

| Componente | Especificação |
|---|---|
| 🧠 **SoC** | Broadcom **BCM2712** |
| 🔢 **CPU** | Quad-core **ARM Cortex-A76** 64 bits @ **2,4 GHz** |
| 🎮 **GPU** | VideoCore VII @ 800 MHz (OpenGL ES 3.1, Vulkan 1.2) |
| 💾 **RAM** | 2, 4, 8 (e até 16) GB LPDDR4X |
| 💿 **Armazenamento** | Cartão microSD |
| 📶 **Wi-Fi** | Dual-band 802.11ac (2,4 e 5 GHz) |
| 🔵 **Bluetooth** | 5.0 / BLE |
| 🌐 **Ethernet** | Gigabit |
| 🔌 **USB** | 2× USB 3.0 + 2× USB 2.0 |
| 📺 **Vídeo** | 2× micro-HDMI (4Kp60 cada) |
| 📷 **Câmera/Display** | 2× MIPI DSI/CSI (4 vias) |
| ⚡ **Expansão** | PCI Express 2.0 (1 via) — permite SSD |
| 📍 **GPIO** | Header de 40 pinos (compatível com versões anteriores) |
| 🔋 **Alimentação** | USB-C, 5V/3A (mín.) ou 5V/5A (completo) |

> A Pi 5 é potente o bastante para servir como **desktop leve**, **servidor doméstico**, central de **mídia** ou cérebro de projetos de **IA na borda**.

<a id="4-raspberry-pi-versus-arduino"></a>
## 4. Raspberry Pi versus Arduino

| Aspecto | 🍓 Raspberry Pi | 🔌 Arduino |
|---|---|---|
| Tipo | Computador (SBC) | Microcontrolador |
| Sistema | Linux completo | Sem SO (firmware) |
| Melhor para | Software, rede, vídeo, IA | Sensores, tempo real, eletrônica |
| Consumo | Maior | Mínimo |
| GPIO | Sim (3,3 V) | Sim (3,3/5 V) |

> 💡 **Complementares, não rivais:** é comum usar um **Raspberry Pi** como "cérebro" e **Arduinos** como "músculos" lendo sensores em tempo real.

<a id="5-onde-o-raspberry-pi-é-usado"></a>
## 5. Onde o Raspberry Pi é usado

- 🖥️ Desktop educacional de baixo custo
- 🏠 Servidores caseiros, *media centers* (Kodi), *Pi-hole*
- 🤖 Robótica e visão computacional
- 🏭 Automação industrial e protótipos
- 🧠 **IA na borda** (*edge AI*) com aceleradores
