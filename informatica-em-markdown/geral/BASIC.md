<a id="1-a-origem-do-basic"></a>
## 1. A origem do BASIC

O **BASIC** (*Beginner's All-purpose Symbolic Instruction Code*) foi criado por **John G. Kemeny** e **Thomas E. Kurtz** em **1964**, no **Dartmouth College**. O objetivo era tornar a programação **acessível** a estudantes e usuários **sem conhecimento técnico avançado** — algo difícil com linguagens da época como **FORTRAN** e **ALGOL**.

- 🖥️ O BASIC original rodava no **Dartmouth Time-Sharing System (DTSS)**, permitindo que **vários usuários** programassem ao mesmo tempo.

---

<a id="2-os-primeiros-interpretadores"></a>
## 2. Os primeiros interpretadores

- 🖥️ Implementado primeiro em **mainframes** e **minicomputadores**, como o **GE-225** e o **PDP-8** (DEC).
- ⚡ Os usuários digitavam programas em **terminais** conectados ao sistema de tempo compartilhado e recebiam **resultados em tempo real** — uma inovação que ajudou a popularizar a linguagem.

---

<a id="3-o-basic-nos-microcomputadores"></a>
## 3. O BASIC nos microcomputadores

Com os **microcomputadores** dos anos 1970, o BASIC virou uma das primeiras linguagens amplamente adotadas:

- 🧩 O **Altair 8800** (1975) usava o **Microsoft BASIC** (originalmente **Altair BASIC**), criado por **Bill Gates** e **Paul Allen** para o processador **Intel 8080**.
- 💰 O **Altair BASIC** foi o **primeiro grande sucesso de software da Microsoft** — e o licenciamento para outras fabricantes consolidou o BASIC como linguagem padrão dos PCs.

> 💡 **Marco da Microsoft:** o BASIC literalmente **fundou** a empresa de Gates e Allen. Antes do MS-DOS e do Windows, era o BASIC que pagava as contas.

> 📜 **A "Carta Aberta aos Hobbistas" (1976):** irritado com a **pirataria** do Altair BASIC (copiado em fitas entre entusiastas), **Bill Gates** publicou uma carta acusando-os de "roubar" software. Foi um marco na ideia de **software como produto pago** — e não algo a se compartilhar livremente.

<a id="4-basic-em-rom-nos-anos-1980"></a>
## 4. BASIC em ROM nos anos 1980

Nos anos 1980, o BASIC vinha **pré-instalado na ROM** de muitos computadores pessoais:

| Computador | Dialeto do BASIC |
|---|---|
| **Apple II** | Integer BASIC / Applesoft BASIC |
| **Commodore 64** | Commodore BASIC |
| **IBM PC** | Microsoft BASIC (Cassette BASIC / BASICA) |
| **TRS-80** | Microsoft BASIC |

> ⚡ O interpretador embutido permitia **começar a programar assim que o computador ligava**, sem sistemas operacionais complexos.

---

<a id="5-evolução-do-basic-versus-os-bits-dos-processadores"></a>
## 5. Evolução do BASIC versus os bits dos processadores

| Era | Bits | BASIC comum | Tipos numéricos |
|---|---|---|---|
| **1970s** | 8 bits | Tiny / Integer BASIC | Inteiros (8–16 bits) |
| **1980s** | 8–16 bits | Microsoft BASIC, GW-BASIC | Inteiros + ponto flutuante 32 bits |
| **1990s** | 16–32 bits | QuickBASIC, QBasic, Visual Basic | `Integer`, `Long`, `Double` |
| **2000s+** | 32–64 bits | Visual Basic .NET | `Integer` (32), `Long` (64), `Decimal` (128) |

**Destaques:**
- 🍎 **Apple I:** Integer BASIC (só inteiros de 16 bits); **Apple II:** Applesoft BASIC (ponto flutuante de 5 bytes ≈ 40 bits).
- 🪟 Nos anos 1990, o **Visual Basic** trouxe programação visual para Windows e compilação para código nativo.
- ⚙️ No **VB.NET**, surgiram tipos dinâmicos e compilação **JIT** (*Just-In-Time*).

<a id="6-principais-dialetos-do-basic"></a>
## 6. Principais dialetos do BASIC

| Dialeto | Ano | Destaque |
|---|---|---|
| **Dartmouth BASIC** | 1964 | O primeiro, no DTSS |
| **Altair BASIC** | 1975 | Primeiro produto da Microsoft (Intel 8080) |
| **Applesoft BASIC** | 1977 | Ponto flutuante, licenciado da Microsoft p/ Apple |
| **GW-BASIC / QBasic** | 1980s–90s | Embutido no MS-DOS |
| **Visual Basic** | 1991 | Programação visual para Windows |
| **VB.NET** | 2002 | Plataforma .NET, JIT |

<a id="7-legado"></a>
## 7. Legado

- 🎓 O BASIC **democratizou a programação**, formando uma geração de programadores nos anos 1980.
- 💰 Foi a **base do império Microsoft**.
- 🧩 Sua filosofia "fácil de começar" ecoa em linguagens modernas voltadas a iniciantes (como **Python** e **Scratch**).
- 😅 **Nem todos gostavam:** o influente cientista **Edsger Dijkstra** chegou a dizer que era "praticamente impossível ensinar boa programação a quem teve contato prévio com BASIC" — uma crítica ao abuso do comando `GOTO` e dos números de linha.
